Student Attendance Tracking System
This project is a simple Student Attendance System made using Python. It helps to store and manage student details like name, course, and attendance percentage. Each student is given a unique ID automatically. The program allows the user to add new students, view all student records, update student information, and calculate the average attendance. It uses basic Object-Oriented Programming concepts like classes and objects. The system runs using a menu where the user can choose what action to perform. This project is useful for beginners to understand how Python works in a real-life example.
Simple Banking Sysytem
This project is a simple banking system created using Python. It includes three main classes: Account, Customer, and Transaction. The Account class handles basic banking operations like depositing money, withdrawing money, and checking the balance. The Customer class stores the customer’s name and connects it with their account details. The Transaction class is used to perform deposit and withdrawal actions automatically. The program shows how money is added or removed from the account and also checks if there are enough funds before withdrawing. Finally, it displays the updated account balance. This project helps beginners understand how Object-Oriented Programming works in a real-life banking example.
